# Do Zero ao Deploy

Este é o aplicativo com que iremos finalizar o curso, atualmente o aplicativo tem as seguintes exigências

  - PHP >= 5.6
  - Composer
  - Pacotes:
    - "pagarme/pagarme-php": "^1.0"
    - "respect/validation": "^1.0"
    - "kevinlebrun/slug.php": "1.*"

O projeto ainda não está finalizado, e seria interessante os alunos clonarem este repositório, implementar suas funcionalidades e enviar os seus prórios repositórios para avaliação, para tanto basta apenas:

- git clone https://github.com/MarlonEtiene/dozeroaodeploy.git
- git remote rm origin
- git remote add origin [seu repositório]
- git push -u origin master

Ideal que todos se cadastrem no Pagar.ME e façam a substituição das chaves necessárias no projeto.
